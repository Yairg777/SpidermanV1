🌌 Holo-Link: Dual-Hand Pro

Next-Gen WebXR Holographic Interface & Multiplayer Duel Arena

Holo-Link is a browser-based augmented reality interface that turns your webcam into a neural link. Using advanced AI hand-tracking, users can interact with 3D objects in zero-gravity space, manipulate matter with two-hand gestures, and compete in high-speed multiplayer duels—all without a headset or external hardware.

🚀 Key Features
🧬 Neural Link Calibration: Real-time, 21-joint skeletal hand tracking via MediaPipe AI.

🛰️ Space-Drift Physics: Objects exhibit Zero-G momentum, lazy rotation, and "fluid" responsiveness.

👐 Advanced Gestures:

Pinch-to-Grab: Use your primary hand to capture and move objects.

Dual-Hand Scaling: Hold an object with one hand and pinch with the second to dynamically resize it in 3D space.

Momentum Throw: Release a pinch while moving to launch objects across the void.

🌐 Optimized Multiplayer: Peer-to-Peer (WebRTC) connectivity via PeerJS. Syncs hand positions and object states with zero-lag data flattening.

⚔️ Duel Mode: A 30-second reflex-based game. Slap falling objects out of the air to score points.

🌈 Winner's Reward: The duel victor gains a "Rainbow Glow" effect on their holographic hand skeleton.

📱 Universal Compatibility: Fully optimized for Mobile (Touch/Camera) and PC.

🎮 How to Use
Initialize: Open the application and click "INITIALIZING NEURAL LINK". Allow camera access.

Interact:

Bring your hands into view.

Pinch (Thumb + Index) near an object to grab it.

While holding, Pinch with your second hand and move them apart/together to scale the object.

Multiplayer:

Host: Click CREATE ROOM, and share the 4-digit code with a friend.

Join: Enter the 4-digit code and click JOIN ROOM.

Duel: Once connected, click the DUEL button on the sidebar to start the 30-second countdown.

🛠️ Built With
Three.js - 3D Engine for rendering the holographic environment.

MediaPipe Hands - AI-powered skeletal tracking.

PeerJS - P2P WebRTC data channels for real-time multiplayer.

Camera Utils - Optimized camera handling for mobile/PC.
